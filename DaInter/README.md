## Install Run Test Build
- npm i
- npm run dev
- npm run test
- npm run build

This setup includes:
* [Vite](https://vitejs.dev/)
* [React](https://reactjs.org/)
* [Redux](https://redux.js.org/)
* [TypeScript](https://www.typescriptlang.org/)
